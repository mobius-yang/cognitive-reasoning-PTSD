# 认知推理 课程项目

项目结构（主要文件）

- src/: 源代码
- data/: 原始/中间数据
- results/: 结果与指标
- pyproject.toml: 项目信息
- requirements.txt: 运行依赖

快速开始

1. 创建虚拟环境并激活（建议）

```bash
python3 -m venv .venv
source .venv/bin/activate
```

2. 安装依赖

```bash
pip install -r requirements.txt
```

3. 运行示例（根据项目需要调整）

```bash
python src/model.py
```
